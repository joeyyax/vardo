# Engagement Wizard Schema

This document defines how proposals, contracts, and addenda are generated.

---

## Step 1: Client
Fields:
- Client legal name
- Primary contact
- Billing contact
- Project name

---

## Step 2: Proposal
Options:
- Create new proposal
- Link existing proposal

If linked:
- Snapshot proposal at time of contract generation

---

## Step 3: Engagement Type
Options:
- Hourly
- Retainer
- Retainer + Additional Hours
- Fixed Scope
- Maintenance Only
- One-Off Task
- Consulting

---

## Step 4: Core Terms (per type)

### Hourly
- Hourly rate
- Billing cadence
- Time increment

### Retainer
- Monthly retainer amount
- Included hours (up to)
- Start date

### Hybrid
- Retainer amount
- Included hours
- Overage rate

### Fixed Scope
- Deliverables
- Fee
- Payment schedule

---

## Step 5: Hosting
Toggle:
- ☐ Hosting provided

If enabled:
- Attach Hosting Addendum
- Attach Data Ownership & Access
- Show Migration Assistance option

If disabled:
- No hosting documents attached

---

## Step 6: Optional Attachments
Toggles:
- ☐ Statement of Work
- ☐ Change Order (later)
- ☐ Support Expectations
- ☐ Responsibility Matrix

---

## Step 7: Review
- Show rendered proposal (if applicable)
- Show rendered contract
- Highlight non-default changes

---

## Step 8: Acceptance
- Online acceptance
- Store signed snapshot
- Trigger onboarding checklist
