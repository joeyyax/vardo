# Request My Application Data

Your application data is always yours.

---

## What You Can Export

You can request an automated export of data associated with your hosted application.

This export includes:
- The application source code as deployed
- A database backup for the application
- Uploaded media and files

---

## What Is Not Included

The automated export does not include:
- Internal Provider systems
- Billing, proposal, or contract records
- Client management or administrative data
- Internal notes or operational metadata

---

## Purpose

This export is intended to give you access to your application data for backup, migration, or archiving.

---

## Timing and Format

Exports are generated based on system availability and reflect the system state at the time of export.

---

## Questions

If you have questions about what’s included, just ask.
