# Working on Multiple Projects

It’s common to have more than one project running at the same time.

Each project:
- Has its own scope
- Has its own timeline
- Moves independently

---

## Why Projects Are Separate

Separating projects helps:
- Keep scope clear
- Avoid confusion about what’s included
- Make progress easy to see

Even when work touches the same site or system, different goals deserve their own space.

---

## What This Means for You

You’ll always be able to:
- See where each project stands
- Know what’s active
- Understand what’s coming next

Nothing gets mixed together.

---

## Summary

Multiple projects mean clarity — not complexity.
