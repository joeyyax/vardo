# Reporting Issues

Reporting an issue is designed to be quick and simple.

You don’t need to explain technical details — just point out what feels wrong.

---

## How to Report an Issue

1. Click “Report an issue”
2. Hover over the page to highlight the problem area
3. Click to select it
4. Add a short description
5. Submit

That’s it.

---

## What Happens Next

Your report is sent directly to the team with helpful context attached.

We’ll review it and:
- Fix it as part of ongoing work, or
- Follow up if clarification is needed, or
- Propose a separate project if it’s out of scope

---

## What Gets Captured Automatically

You don’t need to worry about technical details.

We automatically capture:
- Page URL
- Screenshot
- Browser and device info

---

## Summary

Reporting issues should feel easy.

Just point — we’ll handle the rest.
