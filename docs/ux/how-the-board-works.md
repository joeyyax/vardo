# How the Board Works (Client facing)

The task board shows progress at a glance.

It’s designed to keep things clear and predictable.

---

## What You’ll See

You’ll see:
- What’s planned
- What’s in progress
- What’s waiting
- What’s complete

Each task includes context and a place to comment.

---

## How to Engage

You can:
- Comment on tasks
- Answer questions
- Upload files
- Request new work

If something needs attention, commenting is the fastest way to flag it.

---

## About Task Status

Task status is managed by the team so priorities stay clear and consistent.

You never need to move tasks yourself.

---

## Summary

The board keeps you informed without asking you to manage the workflow.
