# How Tasks Work

Tasks are how work gets done inside a project.

They show what’s planned, what’s in progress, and what’s complete.

---

## What You Can Do

You can:
- View tasks and their status
- Comment and ask questions
- Add new task requests
- Approve or review work when needed

---

## How Status Works

Task status reflects the current state of work.

Status changes are managed by the team so priorities stay clear and predictable.

If something needs attention or feels urgent, commenting is the fastest way to flag it.

---

## When Something Is More Than a Task

If a request introduces:
- New scope
- A new timeline
- Additional cost

We’ll treat it as a separate project so expectations stay clear.

---

## Summary

Tasks give you visibility and a place to engage — without requiring you to manage the workflow.
